# PARSER

## Че делает
Преобразование сырого текста SQL -запроса во внутреннее представление - абстрактное синтаксическое дерево AST.

Просто парсит входящий текст, проверяет грамматику и раскидывает слова по структурам.


### Вход:
`std::string`
### Выход:
`std::unique_ptr<ASTNode>` - указатель на корень AST дерева
