# Storage

## Че делает
Низкоуровневая работа с файловой системой. Чистая работа с байтами.
Сохраняет массивы байт в файлы на диске и читает их обратно по физическому адресу.

### Вход:
Массив байт `std::vector<uint8_t>)`, имя БД, имя таблицы
### Выход:
1) запись: `RecordId` - физический адрес в файле
2) чтение: Считанный массив байт по `RecordId`
